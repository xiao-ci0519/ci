# iPhone界面模拟网页

这是一个模仿真实iPhone手机界面的网页，严格按照像素级尺寸和布局要求实现。

## 功能特点

### 1. 精确尺寸
- 页面整体大小：375px × 812px，模拟iPhone手机屏幕
- 页面四周有14px的内边距
- 状态栏高度：44px
- 底部Dock栏高度：50px

### 2. 界面组件

#### 顶部区域
- 两个圆形头像（60px × 60px），间距12px
- 背景卡片：圆角16px，半透明白色，毛玻璃效果
- 文字输入框：默认文字"我们永远不会败给时间"

#### 中部左侧区域
- 图片占位区：108px × 80px，圆角8px
- 输入框：圆角4px，默认文字"输入文字..."
- 背景卡片：圆角12px，半透明白色

#### 中部右侧区域
- 2×2图标网格（微信、备忘录、知乎、文件夹）
- 图标大小：24px × 24px
- 文字标签：12px字体

#### 中下区域
- 两个聊天输入框：
  - 左侧：160px宽，默认文字"以后...别再流眼泪了"
  - 右侧：180px宽，默认文字"好 希望以后每滴眼泪都是通往幸福"
- 间距：12px
- 距离底部标签栏：48px

#### 底部Dock栏
- 四个应用图标：电话、设置、美化、查手机
- 图标大小：24px × 24px
- 文字标签：10px字体
- 毛玻璃效果：`backdrop-filter: blur(10px)`

### 3. 交互功能
- 所有输入框可编辑（点击编辑，失去焦点恢复默认文本）
- 图标点击有动画效果和通知提示
- Dock栏图标点击有交互反馈
- 状态栏显示实时时间
- 鼠标悬停效果
- 键盘快捷键支持（ESC清除焦点，Ctrl+Enter换行）

## 技术实现

### 核心技术
- **HTML5**：语义化结构
- **CSS3**：
  - Flexbox布局
  - Grid布局（图标网格）
  - 毛玻璃效果（backdrop-filter）
  - 半透明背景（rgba）
  - 阴影效果（box-shadow）
  - 圆角边框（border-radius）
- **JavaScript**：
  - DOM操作
  - 事件处理
  - 动态内容更新
  - 交互反馈

### 第三方资源
- Font Awesome 6图标库（CDN）
- Google Fonts（Inter字体）

## 运行方式
1. 直接双击打开 `index.html` 文件
2. 或使用HTTP服务器（如Live Server）运行

## 浏览器兼容性
- 现代浏览器（Chrome 76+、Firefox 70+、Safari 13+、Edge 79+）
- 毛玻璃效果需要浏览器支持 `backdrop-filter`

## 设计规范
- 颜色方案：iPhone风格浅色主题
- 字体：Inter，苹果系统字体备用
- 间距：精确像素控制
- 阴影：多层阴影增强层次感

## 自定义
可以通过修改以下CSS变量调整界面：
- 主容器尺寸：`.iphone-container` 的 `width` 和 `height`
- 颜色主题：修改 `background-color` 和 `color` 值
- 圆角大小：调整 `border-radius` 值
- 阴影效果：修改 `box-shadow` 值

## 注意事项
1. 毛玻璃效果在某些旧版浏览器中可能不可用
2. 输入框使用 `contenteditable` 属性模拟，实际项目中可替换为 `<input>` 或 `<textarea>`
3. 图标点击功能为演示用途，实际应用中可链接到相应功能
